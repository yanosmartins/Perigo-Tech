Perigo Tech – E-commerce & User Management
O Perigo Tech é um sistema web acadêmico que integra uma plataforma de e-commerce a um robusto gerenciamento de usuários. Desenvolvido com foco em segurança e escalabilidade, o projeto utiliza uma arquitetura modular que separa os ambientes de autenticação e a loja virtual em bancos de dados distintos, otimizando a manutenção e a organização do sistema.

👥 Integrantes
Diogo Bello, Italo Romulo, Luiz Dias e Yan Oliveira.
Localização: Campo Grande – RJ.

🛠️ Tecnologias Utilizadas
Frontend: HTML5, CSS3 e JavaScript.
Backend: PHP.
Banco de Dados: SQL (MySQL).
Biblioteca Adicional: FPDF (para geração de relatórios em PDF).

🚀 Funcionalidades Principais
Módulo de Usuário: Cadastro, login seguro e Autenticação em Dois Fatores (2FA).
E-commerce: Catálogo de produtos e sistema de carrinho de compras.
Painel Administrativo: Gestão completa de usuários e visualização da estrutura do banco de dados.
Segurança e Auditoria: Registro de logs do sistema com opção de exportação em PDF.
Acessibilidade: Suporte a alto contraste e ajuste dinâmico do tamanho da fonte.

🎯 Objetivo do Projeto
Demonstrar a implementação de um ecossistema web completo, explorando conceitos avançados de segurança (logs e 2FA), arquitetura de sistemas e acessibilidade digital.
